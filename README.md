## ✅ 核心特性

1.预设频率挡位 + 自定义频率输入（1~200 Hz）

2.左/右/中键选择

3.全屏半透明区域框选（支持拖拽调整）

4.区域内随机坐标点击 + 实时坐标预览

5.紧急停止：Esc 热键 + 屏幕左上角 FailSafe

6.线程安全 + UI 响应流畅

7.状态提示 + 区域可视化反馈

## ⚠️ 使用前必读

1.仅限合法场景使用（如游戏挂机需确认游戏规则允许）

2.首次运行建议在空白桌面测试

3.紧急停止：按 ESC 键 或 将鼠标移至屏幕左上角（10×10 区域）

4.Windows 需以管理员身份运行（部分系统权限要求）

## 📦 安装依赖

```
pip install pyautogui PyQt5 keyboard
```
---
**注：keyboard 用于全局热键监听（Esc 停止），Linux/macOS 可能需额外权限。**
